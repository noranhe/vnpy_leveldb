## 说明

为vn.py提供了LevelDB数据库的使用支持

## 安装

安装需要基于2.4.0版本以上的[VN Studio](https://www.vnpy.com)。

安装需要基于leveldb数据库

下载解压后在cmd运行：

```
python setup.py install
```
